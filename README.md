# Personal Blog Web App (Node.js & EJS)

This repository contains my completed backend-driven web solution to the [Personal Blog](https://roadmap.sh/projects/personal-blog) challenge on roadmap.sh, featuring dynamic server-side rendering, session authentication, and automated date tracking.

## Features
- **Express & EJS Templating Engine**: Server-rendered views (`home`, `admin`, `article`, `new`, `edit`, `login`) managed cleanly with dynamic EJS templates.  
- **Session-Based Admin Dashboard**: Secure authentication middleware protecting article creation, updates, and deletion behind `/admin`.  
- **Automated Date Management**: Creation and update timestamps (`YYYY-MM-DD` and `editedAt`) are automatically generated by the server and displayed in read-only form controls.  
- **Adaptive UI & Design System**: Minimalist interface featuring dark/light theme switching (`data-theme="dark"`), dynamic back navigation, and localized styling utilities.

## Requirements Met
- **Full CRUD Functionality**: Create, read, update, and delete operations fully implemented for articles using persistent JSON file storage.
- **Route Protection & Authentication**: Sensitive routes (`/admin`, `/new`, `/edit/:id`, `/delete/:id`) strictly guarded by Express session middleware (`requireAuth`).
- **Context-Aware Navigation**: Integrated origin detection so viewing an article dynamically routes users back to their previous page (`/admin` vs `/home`).
- **Input Validation & Consistency**: Standardized form styling with read-only fields for immutable metadata like publication dates.
- **Error Handling & UX Alignment**: Custom error placements, session cleanup on re-login, and intuitive header placement for global site navigation.

## How to Run
1. Make sure you have **Node.js** installed.
2. Clone or download the repository and install dependencies:
    ```bash
    npm install
    ```
3. Create a `.env` file in the root folder with your credentials:
   ```code snippet
    PORT=3000
    SESSION_SECRET=your_secret_key
    ADMIN_USER=your_admin_username
    ADMIN_PASS=your_admin_password
   ```
4. Start the server using your terminal:
    ```bash
    node server.js
    ```
4. Open your browser and navigate to (http://localhost:3000).
